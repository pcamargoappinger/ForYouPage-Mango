Einrichtung

Einmalig einrichten – danach ist dein Feed für alle via QR-Code erreichbar.

Schritt 1 von 3 – Feed konfigurieren
Schritt 2 von 3 – Datenspeicher einrichten

Damit Beiträge auf allen Geräten sichtbar sind, wird ein kostenloser Datenspeicher bei JSONBin.io verwendet.

1. Gehe auf jsonbin.io und registriere dich (kostenlos)
2. Klicke oben rechts auf deinen Namen → API Keys
3. Kopiere den Master Key (beginnt mit $2a$10$…)
4. Füge ihn unten ein und klicke auf „Weiter"
Schritt 3 von 3 – Fertig!
✅ Dein Klassen-Feed ist einsatzbereit!

Im Admin-Bereich findest du den QR-Code, den du im Unterricht zeigen kannst. Schüler:innen sehen beim Scan sofort den Feed.
